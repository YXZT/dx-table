## dx-table

这是基于Naive UI的数据表格Data Table组件进行二次封装的表格组件，是个人项目中的一种实践

- 使用TS
- 兼容所有原本的[表格功能](https://www.naiveui.com/zh-CN/os-theme/components/data-table#API "表格功能")，包括虚拟滚动
- 自定义列的显示隐藏
- 自定义列的fixed位置
- 改变列的顺序
- 配置持久化存储
- 封装请求逻辑简化代码
- 无限滚动的支持
- 对单选/多选更好的支持
- 显示已勾选的数量
- 键盘快捷键
- 表格动态添加行
- 表格动态弹出层
- 表格编辑状态与非编辑状态
- 表格导出excel封装，可配置要导出的列
